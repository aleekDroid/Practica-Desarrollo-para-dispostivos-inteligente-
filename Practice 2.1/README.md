# Clima Actual

Una aplicación movil desarrollada en Flutterque muestra el clima (de manera estática).

Este proyecto es una interfaz gráfica hecha con Flutter para mostrar información del clima. Actualmente muestra la temperatura, la ciudad (Santiago de Querétaro) y un ícono que representa el estado del tiempo.

## Características
* Muestra la temperatura en grados centigrados y la ubicación.
* Funciona en dispositivos Android.

## Tecnologías utilizadas
* Flutter: Framework para el desarrollo de la interfaz.
* Dart: Lenguaje de programación utilizado.

## ¿Cómo ejecutar el proyecto?

1. Se abre la terminal en la carpeta del proyecto.
2. Ejecuta el siguiente comando para revisar las dependencias:
   flutter pub get.
3. Inicia la aplicación con el siguiente comando:
   flutter run.

## Resultado.
Se muestra la app con el ctema de material design.
EL hotload funciona con ctrl+s.
